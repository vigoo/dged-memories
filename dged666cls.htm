A CLS plugin architektúra sem volt még kiforrott, de a wall settingset felváltotta a wall class plugin. Az engine hiányosságai miatt ekkor még nem lehetett további objektumokhoz plugin írni, ezért ez inkább a későbbi DGed-ek alapjának tekinthető, azok szinte nagyon nagy mértékben a CLS pluginokra épültek. A class settings dialog itt még modális, OK és cancel gombbal ellátva, amiből a cancel nem működött. A Base fül minden classhez közös volt (bár úgyis csak a wall class volt még). Itt az engine base class flagjeit lehett állítani, valamint egy a classek közötti konverzióra való combo-box is megjelent, ami aztán később megszűnt, itt pedig nem csinált semmit.
